# .tsv及.csv格式文件

**TSV：tab separated values；即“制表符分隔值”，如：**

```
name    age
张三      18
李四      28
```

**TSV：tab separated values；即“制表符分隔值”，如：**

```
name,age
张三,18
李四,28
```

> 两者均是以纯文本形式存储的表格数据，可由表格软件导出或读入，编码方式是可以自行定义的。本质而言没有太大区别。
